{% if code == "BILIRUBIN" %}
Interpretation:
  • Bilirubin is a yellow pigment produced when red blood cells break down. It is conjugated by the liver and excreted in bile.
Increased Total Bilirubin / Hyperbilirubinemia:
  • Hemolytic anemia (raised Indirect Bilirubin).
  • Viral, alcoholic or drug-induced hepatitis.
  • Cirrhosis of the liver.
  • Bile duct obstruction — gallstones, tumours (raised Direct Bilirubin).
  • Gilbert's syndrome, Crigler-Najjar syndrome.
  • Physiological / pathological neonatal jaundice.
Decreased Bilirubin:
  • Generally not clinically significant.
{% elif code == "CHOL" %}
Interpretation:
  • Total Cholesterol is a key marker of cardiovascular risk. ATP-III categories: Desirable < 200 mg/dL, Borderline-High 200–239, High ≥ 240.
Increased Cholesterol:
  • Atherosclerosis, coronary artery disease.
  • Familial hypercholesterolemia.
  • Hypothyroidism.
  • Nephrotic syndrome, obstructive jaundice.
  • Diabetes mellitus, obesity.
  • High saturated-fat / sedentary lifestyle.
Decreased Cholesterol:
  • Severe liver disease.
  • Malabsorption / malnutrition.
  • Hyperthyroidism.
{% elif code == "TG" %}
Interpretation:
  • Triglycerides are the most common form of fat in the body. ATP-III categories: Normal < 150 mg/dL, Borderline 150–199, High 200–499, Very High ≥ 500.
Increased Triglycerides:
  • Diabetes mellitus, metabolic syndrome.
  • Obesity, sedentary lifestyle.
  • Alcohol abuse.
  • Hypothyroidism, nephrotic syndrome.
  • Familial hypertriglyceridemia.
  • Acute pancreatitis (TG > 500 mg/dL).
Decreased Triglycerides:
  • Malabsorption, malnutrition.
  • Hyperthyroidism.
{% elif code == "HBA1C-T" %}
Interpretation:
  • HbA1c reflects average plasma glucose over the past 8–12 weeks and is the gold standard for monitoring glycaemic control.
ADA Categories:
  • Non-diabetic: < 5.7%
  • Prediabetes: 5.7–6.4%
  • Diabetes: ≥ 6.5%
  • Therapeutic target for most diabetics: < 7%
Falsely Low HbA1c:
  • Hemolytic anemia, recent blood loss or transfusion.
  • Hemoglobinopathies (HbS, HbC).
Falsely High HbA1c:
  • Iron-deficiency anemia.
  • Splenectomy, uremia.
{% elif code == "VITD-T" %}
Interpretation:
  • Vitamin D 25-Hydroxy is the storage form and the best marker of vitamin D status.
Status Categories:
  • Deficient: < 20 ng/mL
  • Insufficient: 20–29 ng/mL
  • Sufficient: 30–100 ng/mL
  • Toxic: > 150 ng/mL
Causes Of Deficiency:
  • Inadequate sun exposure, dark skin.
  • Malabsorption (celiac, IBD, gastric surgery).
  • Chronic kidney or liver disease.
  • Causes rickets in children and osteomalacia in adults.
{% elif code == "VITB12-T" %}
Interpretation:
  • Vitamin B12 is essential for red-cell maturation, DNA synthesis and neurological function.
Decreased Vitamin B12:
  • Pernicious anemia (intrinsic-factor deficiency).
  • Strict vegetarian / vegan diet.
  • Atrophic gastritis, gastric bypass, ileal resection.
  • Inflammatory bowel disease, fish-tapeworm infection.
  • Long-term metformin or PPI therapy.
  • Causes megaloblastic anemia and peripheral neuropathy.
Increased Vitamin B12:
  • Liver disease, myeloproliferative disorders.
{% elif code == "URINE-CT" %}
Interpretation:
  • Urine culture identifies the organism causing a urinary tract infection (UTI) and guides antibiotic therapy.
Significance of Growth:
  • ≥ 105 CFU/mL of a single organism — significant.
  • 104–105 CFU/mL — possibly significant; correlate clinically.
  • < 104 CFU/mL or mixed flora — likely contamination.
  • "No growth after 48 hours" — sterile sample.
Common Pathogens:
  • Escherichia coli (most common).
  • Klebsiella, Proteus, Pseudomonas.
  • Enterococcus, Staphylococcus saprophyticus.
{% elif code == "HB-T" %}
Interpretation:
  • Haemoglobin (Hb) carries oxygen in red blood cells. Its level is the primary screen for anemia and polycythemia.
Decreased Hb / Anemia:
  • Iron, B12 or folate deficiency.
  • Anemia of chronic disease, chronic kidney disease.
  • Acute or chronic blood loss.
  • Hemolysis, thalassemia, sickle cell disease.
  • Bone-marrow failure (aplastic anemia, leukemia).
Increased Hb:
  • Polycythemia vera.
  • Chronic hypoxia — COPD, smoking, high altitude.
  • Dehydration (relative increase).
{% elif code == "UA-T" %}
Interpretation:
  • Uric acid is the end-product of purine metabolism. Persistently raised levels predispose to gout and uric-acid stones.
Increased Uric Acid / Hyperuricemia:
  • Gout.
  • Renal failure, dehydration.
  • High-purine diet — red meat, organ meat, seafood, alcohol.
  • Thiazide / loop diuretics, low-dose aspirin.
  • Tumor lysis syndrome, leukemia, lymphoma.
  • Hypothyroidism, hyperparathyroidism.
Decreased Uric Acid:
  • Wilson's disease, Fanconi syndrome.
  • SIADH, allopurinol therapy.
{% elif code == "BG-T" %}
Interpretation:
  • ABO and Rh(D) typing identifies the patient's major red-cell antigens — essential before transfusion, surgery and during pregnancy.
ABO System:
  • Group A — A antigen on RBC, anti-B in plasma.
  • Group B — B antigen on RBC, anti-A in plasma.
  • Group AB — both antigens, no antibodies (universal recipient).
  • Group O — neither antigen, both antibodies (universal donor).
Rh System:
  • Rh-positive: D antigen present.
  • Rh-negative: D antigen absent.
  • Rh-negative mothers carrying Rh-positive babies need anti-D prophylaxis to prevent hemolytic disease of the newborn.
{% elif code == "CA-T" %}
Interpretation:
  • Serum calcium reflects the balance between bone, kidneys, parathyroid hormone and vitamin D.
Increased Calcium / Hypercalcemia:
  • Primary hyperparathyroidism.
  • Malignancy — multiple myeloma, bone metastases, PTHrP-secreting tumours.
  • Vitamin D toxicity, milk-alkali syndrome.
  • Sarcoidosis, granulomatous disease.
  • Thiazide diuretics.
Decreased Calcium / Hypocalcemia:
  • Hypoparathyroidism, post-thyroidectomy.
  • Vitamin D deficiency, rickets, osteomalacia.
  • Chronic kidney disease.
  • Acute pancreatitis.
  • Hypoalbuminemia (correct for albumin).
{% elif code == "SGPT-T" %}
Interpretation:
  • SGPT (ALT) is a liver-specific enzyme. A rise indicates hepatocellular injury.
Increased SGPT:
  • Viral hepatitis (A, B, C, E).
  • Non-alcoholic and alcoholic fatty liver disease.
  • Drug-induced hepatitis — paracetamol overdose, statins, antitubercular drugs.
  • Autoimmune hepatitis, Wilson's disease, hemochromatosis.
  • Cirrhosis (often only modestly raised).
  • Right-heart failure (congestive hepatopathy).
{% elif code == "SGOT-T" %}
Interpretation:
  • SGOT (AST) is found in liver, heart muscle, skeletal muscle, kidney and red blood cells. A rise reflects tissue injury but is less liver-specific than ALT.
Increased SGOT:
  • Hepatitis, cirrhosis, fatty liver.
  • Acute myocardial infarction.
  • Skeletal-muscle injury, rhabdomyolysis.
  • Hemolytic anemia.
  • AST/ALT ratio > 2 strongly suggests alcoholic liver disease.
{% elif code == "GGT" %}
Interpretation:
  • Gamma-Glutamyl Transferase (GGT) is the most sensitive enzyme for detecting cholestasis, biliary disease and chronic alcohol use. It rises before ALP and stays elevated longer.
Increased GGT:
  • Chronic alcohol abuse — most sensitive single marker.
  • Biliary obstruction (gallstones, tumours, primary biliary cholangitis).
  • Intra-hepatic cholestasis — drugs, sepsis, pregnancy.
  • Acute and chronic hepatitis, cirrhosis, fatty liver.
  • Drug induction — phenytoin, barbiturates, oral contraceptives, rifampicin.
  • Pancreatitis, pancreatic cancer.
  • Congestive heart failure, acute myocardial infarction.
Clinical Use:
  • An isolated raised ALP with normal GGT points to bone, not liver, origin.
  • Persistent elevation despite abstinence is a marker of poor prognosis in alcoholic liver disease.
{% elif code == "LDH" %}
Interpretation:
  • Lactate Dehydrogenase (LDH) is a non-specific marker of cellular injury — present in liver, heart, skeletal muscle, kidneys, brain, lungs and red blood cells.
Increased LDH:
  • Hemolytic anemia, megaloblastic anemia (often markedly raised).
  • Acute myocardial infarction, congestive cardiac failure.
  • Hepatitis, cirrhosis, hepatic congestion.
  • Pulmonary embolism, pulmonary infarction.
  • Skeletal-muscle injury, rhabdomyolysis, muscular dystrophies.
  • Lymphoma, leukemia, germ-cell tumours — high cell turnover.
  • Renal infarction, acute kidney injury.
Clinical Use:
  • Marker of disease activity and prognosis in lymphoma and germ-cell tumours.
  • LDH/AST ratio > 5 suggests hemolysis rather than hepatocellular injury.
{% elif code == "DEN-NS1" %}
Interpretation:
  • Dengue NS1 (Non-Structural protein 1) antigen is the earliest marker of dengue infection — detectable from Day 1 of fever, persisting until Day 5–7. It allows acute diagnosis before antibodies appear.
Result Interpretation:
  • Positive — Acute dengue infection (current viraemia).
  • Negative after Day 7 — does not exclude dengue; follow up with IgM / IgG serology.
  • Equivocal — repeat in 24–48 hours.
Notes:
  • Sensitivity 60–90% in primary infection; lower in secondary infection due to immune-complex masking.
  • Combine with platelet count, hematocrit and clinical assessment for severity grading.
  • Warning signs of severe dengue: persistent vomiting, abdominal pain, bleeding, lethargy, hepatomegaly, rising HCT with falling platelets.
{% elif code == "TSH" %}
Interpretation:
  • TSH is the most sensitive single test for thyroid dysfunction. It is secreted by the anterior pituitary and stimulates the thyroid to produce T3 and T4. Pregnancy-specific reference intervals apply.
Pregnancy & Cord Blood Reference:
  • First Trimester: 0.24–2.99 µIU/mL
  • Second Trimester: 0.46–2.95 µIU/mL
  • Third Trimester: 0.43–2.78 µIU/mL
  • Cord Blood: 2.3–13.2 µIU/mL
Increased TSH (Primary Hypothyroidism):
  • Hashimoto's autoimmune thyroiditis (commonest cause).
  • Iodine deficiency, post-thyroidectomy, post-radioiodine.
  • Drugs — lithium, amiodarone, antithyroid drugs.
  • Subclinical hypothyroidism (TSH raised, T4 normal).
  • Recovery phase of non-thyroidal illness.
Decreased TSH (Hyperthyroidism):
  • Graves' disease, toxic multinodular goitre, toxic adenoma.
  • Subacute / silent thyroiditis (early phase).
  • Excess thyroxine therapy.
  • Secondary / tertiary hypothyroidism (pituitary or hypothalamic — TSH inappropriately low with low T4).
  • Severe non-thyroidal illness ("sick-euthyroid").
{% elif code == "PHOS-T" %}
Interpretation:
  • This will give an idea of renal and bone diseases.
Increased Phosphorus Or Hyperphosphatemia:
  • Renal diseases with increased blood urea (BUN) and creatinine.
  • Hypoparathyroidism with raised phosphate and decreased calcium. But renal function will be normal.
  • Liver diseases and cirrhosis.
  • Acromegaly.
  • Increased dietary intake.
  • Sarcoidosis.
  • Acidosis.
  • Hemolytic anemia.
Decreased Level Of Phosphorus Or Hypophosphatemia:
  • Decreased intestinal absorption.
  • Rickets (Vit. D deficiency).
  • Vomiting and severe diarrhea.
  • Severe malnutrition and malabsorption.
  • Acute alcoholism.
